## Project Description

OpenScene is an AI showrunner workspace for producing short-form video concepts and scenes with Qwen Cloud. A creator starts with a brief, then the app uses Qwen-powered agents to plan scripts, hooks, influencer/background directions, storyboard frames, visual assets, motion-control prompts, and video generation tasks. The project then moves into a workflow graph and timeline editor so the user can review generated assets, assemble scenes, manage aspect ratios, and prepare an export.

The backend uses Qwen Cloud and Alibaba Cloud DashScope APIs for multimodal planning, image generation, Wan video generation, motion-control generation, media upload resolution, and async task polling.
